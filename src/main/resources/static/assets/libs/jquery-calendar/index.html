sys-calendar.js

jquery日历插件,自定义点击事件，记录信息，编辑假日安排等方法，使用简单，更新1.0版本，不定期更新版本，如需获取最新版本，请关注本网站WWW.husysui.com

说明
该插件还使用了sysui.js框架插件进行配置，该js具体说明在以后会发布，请关注我的网站
基础引用
引入3个文件，一个css样式文件，两个js文件一个jquery库，另外一个就是日历插件的js
